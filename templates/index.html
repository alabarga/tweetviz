What should I expect to see?

As you move around in the 3-dimensional space, each point is a tweet of the given user. The points close to each other tend to have similar "semantic", different clusters of points might represent different "topics" in user's tweets.

Note that this "semantic" is computed by skip-thought, which might or might not make a lot of sense. Given the short amount of text in tweets, it is often not very clear how the tweets are clustered. However, there might be some patterns, for instance you might find groups of retweets, groups of tweets containing links, tweets starting with an @-mention, etc..

How does this work?

Nothing actually works. Those points are all randomly generated.

Just kidding. It does work.
Essentially, each tweet is transformed into a 4800 dimensional feature vector using skip-thoughts, those features are then projected into 3-dimensional space using t-SNE. Everything is computed when you hit "Go". We only collect at most 200 latest public tweets. The final result is cached.
